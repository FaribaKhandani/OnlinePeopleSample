## About The Project



In this Application, a list of people's information is shown, and then clicking on a specific user the detail of user's is displayed; 


Which technology:
* Clean architecture which is SOLID principles  MVVM and Kotlin
*  Navigation, Hilt for Dependency Injection
*  Retrofit for Restful API

* [Kotlin](https://github.com/JetBrains/kotlin)
* [MVVM](https://developer.android.com/jetpack/guide)
